# AppIconGen

AppIcon generator for iOS project.



# Installation

`brew install gb`

## cocoapods

```
pod 'AppIconGen'
```

# Usage

build phase.

```
$PODS_ROOT/AppIconGen/appicongen --input $SRCROOT/$PRODUCT_NAME/Artwork.png --xcassets $SRCROOT/$PRODUCT_NAME/Assets.xcassets/
```
